Convert Text To QR Code In PHP Easily

Do you have an application in PHP and you want to convert a string to a QR code? PHPQR is right for you.
Sample Code
include "path/to/phpqr/class.php";
$phpqr = new PHPQR();
$text = "Hello";
$response = $qr->generateQRWithArray("200x200", $text);
echo "QR Code:
";
echo $response["htmlCode"];
